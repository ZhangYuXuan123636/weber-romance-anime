# 免责声明
这是框架的第三版，修改了很多东西，框架分为四部分
第一部分：关系是如何被绑定的（解释这两个人为什么在一起）
第二部分：冲突再生产
第三部分：这段关系在说服谁
第四部分：这段关系是如何变化的

**框架的适用范围**
这个框架仅限于以恋爱为主线（主要冲突）或者有完整明确的恋爱故事的番剧


# 关系是如何被“绑定”的
**这个分类本身是一种理想模型，在实践中一般是多种分类的混合展现**
这里借用了马克斯韦伯的政治合法性模型中的名词，原因是我的灵感来自于他的框架，所以我沿用的原先的名词

## 传统型
**核心逻辑：** 关系源于将过去产生的羁绊运用在现在（在社会学原版中，“传统”也包括了既定的社会规范和习俗，在中国的故事中，比较喜欢将外在法理和传统结合）

**恋爱番表现：** 
青梅竹马从小一起长大，拥有共同的回忆和深厚的羁绊。这种回忆相当于一种信息差，为什么传统型的角色会获胜呢？因为我了解你，我和你有过去的回忆，其他人没有


## 魅力型
**核心逻辑：** 关系源于一方或双方超凡的个人魅力（如外貌、才华、性格等），产生强烈情感吸引。

**恋爱番表现：** 
我因为某个事物或者特质被你吸引，开始注意你，并且加深我们两人之间的情感，魅力型是最常见的，因为这种关系最容易发展故事，同时也最容易被观众所接受


## 法理型
**核心逻辑：** 关系源于明确的规则、契约或理性计算，以及内在的承诺

**恋爱番表现：** 
外部因素使两人不得不相处（规则、责任），比如说先婚后爱、个人责任

### 外化法理型权威
**核心逻辑：** 本质属于法理型权威的亚种，关系的推进源自于明确的规则，外部的契约
**恋爱番表现：** 
大多源自于某些小概率事件，导致两人产生了必然联系，比如说上下级、父母再婚

### 内化法理型（存在主义法理型）
**核心逻辑：** 这个是极度内化的特殊法理型权威，逻辑来自于个人责任（我对个人责任的理解是，一种基于社会道德给自己施加的理性规则约束）
**恋爱番表现：** 
它的表现形式及其特殊，举个例子，比如说galgame（虽然galgame并不适用于这个框架），galgame的恋爱合法性是基于游戏本身在迁就你，你选A或者B，都是平等的，但是朋也的情况十分特殊，他完全可以离开小镇，渚也让他离开小镇，观众也能接受他离开小镇，但是最后，他还是选择了留下来

朋也并没有选择“最优解”，而是选择承担现实

## 总结
事实上，可以将这三个部分比作一场赛跑，传统型有着更前的起跑线，魅力型就是跑步这个行为本身，外部法理型是从终点往回跑，内化法理型是开枪那个

```
这里举一个特例，《败犬女主》，第一部分的本质是分析恋爱关系的合法性，但是败犬女主是建立在合法性被剥夺之后的，不过第二部分开始是生效的

作者采用了情绪余震型、成长型、日常扰动型，主要目的是用日常扰动作为情绪余震的容器，防止变成严肃文学

但是话又说回来，其实可以扩充一下这个框架的，添加一个合法性的竞争与消解，专门用于解释党争类，但是我懒，就先这样把
```

在分析党争类型的番剧时，合法性来自于谁更加合法，而不是两者之间是否存在合法性

# 冲突再生产
我将动漫中会出现的冲突分成三种
## 冲突从哪里来？
### 外在障碍型
冲突一般来自于外在因素
身份、阶级、距离、疾病、家庭 / 社会压力

这种类型是最安全的

### 内在缺陷型
冲突来自于角色自身
性格缺陷、价值观冲突、情感回避、自我否定
·
许多人物驱动的作品就是这部分


### 关系定义型
冲突来自于两人的立场或者身份本身是冲突的
博弈、身份对立、权力不对等、角色互为工具（我需要你，是因为我需要你帮我做某件事）

这种关系是最危险的，因为冲突一旦消失，这段关系就无法被定义（单独提一嘴《NANA》），这种关系天生具有告白即死和转型困难的问题


## 冲突靠什么反复发生？
### 博弈型（特意标注一下，这里的博弈型不是指来源，而是指冲突的表现形式）
什么叫博弈型表达方式呢，意思就是把对话改成“直接说实话”，剧情会立刻结束，那就是博弈型表达

攻防转换、误判、反转、心理优势更替

### 日常扰动型
就是单纯的日常，没什么好说的（《高木》《月色真美》都是这种）

上学、放学、吃饭、回家、偶尔一点小情绪

### 任务 / 事件驱动型
角色之所以在说话，是因为某些任务，有一个目标或者事件短暂的将角色绑在一起

单独说明一下，这种任务驱动本身并不代表感情，之所以有这个任务，目的只是让角色在完成任务的途中展现出性格、价值观、能力

## 冲突为什么结束不了？
为什么角色知道了事件的结果，却还在受到事件影响

### 不可撤回状态（情绪余震型冲突）
这个概念比较重要，我单独讲

一个事件驱动型是怎么发展的
1. 有一个问题
2. 解决它
3. 结束
事件型驱动需要解决接下来怎么办

而不可撤回状态讲述的是，我已经做了这件事，我以后怎么办

你伤害了一个人，并且向对方道歉，对方也接受了，虽然外在冲突被解决，但是角色与角色之间的关系已经被永久性改变（比如说英梨梨）





# 这段关系是为了谁服务
事实上，这种亲密关系的生成是为了两种人服务的，一种是剧情内的角色本身，一种是屏幕外的观众，这两种只能有一个重点（非常重要，本质上是零和博弈，如何把控这个天平异常重要）

### 为角色服务
在这种结构中，重点是角色的体验，主要注重于角色的成长以及情侣角色之间的关系

这种结构的大多出现于传统型和内化法理型
特点是合理性 > 情绪强度

这种对于观众来说没那么好磕，但对于作者来说，十分好写，剧情很难写崩


### 为观众服务
这种结构中，重点是观众的感受，观众是否想看你们在一起？角色是否有为观众服务的“高光时刻”？

这种结构大多出现于魅力型
特点是情绪强度 > 合理性

对于观众来说，这种结构的故事看起来更爽，但是更加考验作者水平，结构容易断裂

### 总结
一个故事可以同时为两边服务，但是最终只能以一种为主
在此，我会举一个正例和反例

```
正例：《高木同学》
高木同学的重点在于服务观众，让观众觉得好磕，剧情内其实没什么复杂的逻辑

反例：《伪恋》
剧情内主要讲述小野寺和千棘的故事（传统+法理），为了角色服务，但是万里花人气太高，导致作者被观众绑架，两边都想写，最终故事失控
```



# 这段恋爱关系是如何变化的

这是最重要的部分，因为它尝试解决前面三个部分都解决不了的问题

第一部分：解释“为什么在一起”
第二部分：解释“现在靠什么爽”
第三部分：解释“写给谁看”

这三个部分都默认剧情的核心爽点是永恒的
那么，这真的是永恒的吗？

我所使用的方法是把剧情核心抽出来，然后提三个问题
他们还会不会对话？对话还有没有内容？关系还有没有推进方式？
如果这个剧情核心被抽出来之后，恋爱关系无法被推进，那么就说明恋爱关系是失败的

假设有一部恋爱番，男主和女主每一集都在打游戏，谁赢了谁就可以让对方做一件事

第一部分：为什么在一起？
因为一起打游戏（外在法理型）

第二部分：冲突怎么来？
输赢（博弈）

第三部分：写给谁看？
观众

第四部分实际上是在回答：如果他们不打游戏了，恋爱还能不能继续作为这部作品的核心？
情况1：剧情完全依靠游戏推进，不打游戏两人完全没有任何交流，那么在这种情况下，一旦失去游戏，关系就会暴死
情况2：游戏作为故事的开头，两人除了游戏也有其他的交流（可能是一起回家），那么在这种情况下，即便失去游戏，两人的关系也能依靠日常推进



## 特别标注
这里说的剧情核心是指推进恋爱关系的剧情核心，比如说《辉夜大小姐》依靠博弈推进恋爱关系，高木依靠捉弄推进恋爱关系

我发现有些人在分析时会下意识的帮自己喜欢的作品圆场，用于证明自己的作品没有问题，为此，我写了三条原则

### 禁止角色重写
“如果两人能坦率一些……”
“如果能开朗一些……”

这些话的本质都是在使用潜在可能性去替代已完成设定
分析时，只允许使用作品已经写出来的那个人


### 禁止题材漂移
“虽然恋爱不好写了，但变成日常番也挺好看的”
“不谈恋爱了，人物互动还是很有趣”

这些话的本质是在转移重心
因为这一部分是在测试恋爱能不能作为主线引擎，而不是作品能不能存在

| 原本问题      | 偷换概念      |
| --------- | --------- |
| 恋爱还能否作为主线 | 这部作品还能否好看 |
| 情感推进是否成立  | 人物还能否互动   |
| 关系是否持续发展  | 世界是否还能运转  |


### 禁止引擎替换 
“换一种方式推进恋爱也行吧？”

这句话的本质是默认“推进方式是可自由替换的”，在结构分析中，这是错误的，一个恋爱故事，只能有一个“主推动引擎”（可以有辅助引擎）

总不能说车子发动机坏了，我们推着车走也算开车吧







# 结论
我将使用龙与虎来讲述这个框架如何使用

首先，关系是如何被绑定的，答案是外在法理型，两人为了帮助对方追人，所以建立起关系

第二，两人的冲突属于什么类型，答案是障碍型（表象）和成长型（内核），两人喜欢的人都是第三者，自己并不是被选择的对象（障碍型），在帮助对方追人的时候，双方都开始不断暴露自己的缺陷，两人的冲突来自两人都不能离开对方，但是不敢承认

第三，这段关系为谁服务，答案明显是角色，龙与虎中使用了大量内心独白表述角色成长

现在得到了三个答案，那么我们将抽离出龙与虎发展的核心，然后思考三个问题
他们还会不会对话？
对话还有没有内容？
关系还有没有推进方式？

- 特别标注：去掉核心的目的并不是看这部番是不是变得难看，而是看逻辑能不能继续推进，剧情还能不能继续跑

他们还会不会对话？
会，因为两人的关系实际上并不是依靠帮对方谈恋爱推进的，而是因为大河就住在龙儿隔壁，大河生活不能自理，龙儿看不下去，然后邀请大河来自己家吃饭

对话还有没有内容？
有，即便不存在恋爱作战会议，两人也可以讨论日常和家庭（大河的父亲、龙儿的妈妈）

关系还有没有推进方式？
有，只不过故事从原先的外在契约型变成了单纯的传统型，并且两人的性格缺陷（大河的暴戾与依赖，龙儿的过度奉献与自我压抑）将更早、更直接地发生摩擦与磨合
