# 免责声明
这个框架问题很多很大，只是我随手写下来的，只能被用于分析需要证明恋爱合法的番剧，比如说有一部战斗番，男女主就是在一起了，你也别管为什么，这种番就不能套用这个框架（意思就是只能分析恋爱作为问题的故事，不能分析恋爱作为状态的故事）

但是反过来说，有一部番，谈恋爱就是明确的主线，那么此时就可以使用这个框架分析恋爱关系是否合法

比如说《败犬女主》，这部番从一开始就不是在竞争合法性，而是在讲一群人失去合法性之后的故事，她们不是输在最后，而是一开始就已经输了（当然了，如果哪天温水突然转性了，剧情开始转向恋爱，那另说）

比如说《伪恋》，从一开始，伪恋的核心就明确为男主的真命天女是谁，女主们争夺的就是谁有更高的合法性

# 关系合法性来源
逻辑上来说，恋爱番中关系的合法性起源可以分成三个部分

## 传统型权威
**核心逻辑：** 合法性源于“历来如此”的习惯与传统。

**恋爱番表现：** 青梅竹马从小一起长大，拥有共同的回忆和深厚的羁绊。这种恋爱的“合法性”不依赖于强烈激情，而是基于时间积淀的稳定感与熟悉感。

**例子：**《名侦探柯南》中的工藤新一与毛利兰（我会根据漫画结局改变我的打法），《哆啦A梦》中的大雄与静香

## 魅力型权威
**核心逻辑：** 合法性源于一方或双方超凡的个人魅力（如外貌、才华、性格等），产生强烈情感吸引。

**恋爱番表现：** 角色在某个契机下相遇，因瞬间的心动或逐渐发现的独特魅力而坠入爱河。这种关系充满戏剧性与不确定性，往往伴随着“命运感”或“一见钟情”。魅力型恋爱合法性依赖于持续的情感投射与认同，若魅力减退或遭遇挫折，关系可能动摇。

**例子：**《辉夜大小姐想让我告白》中的四宫辉夜与白银御行（吃你的炒面去吧），《青春猪头少年不会梦到兔女郎学姐》中的梓川咲太与樱岛麻衣

```
辉夜大小姐是我不得不骂的一环，并且，我会将辉夜大小姐和高木同学做比较

韦伯指出，魅力型权威是不稳定的，它必须不断创造奇迹或胜利来维持。一旦魅力型领袖开始过“柴米油盐”的日子，其神性就会消失，这被称为“魅力的例行化”（这段的原意是用来讲述政权的，我借来用一下）

番外篇中，两人都不再是天才，变成了纯粹的普通人
```

## 法理型权威
**核心逻辑：** 合法性源于明确的规则、契约或理性计算，以及内在的承诺（就比如后面会讲到的中国法理型）

**恋爱番表现：** 在恋爱番中，这种类型较少作为浪漫主线，但常以特殊形式出现，如政治联姻、假恋爱契约、相亲活动或恋爱游戏规则。关系的基础不是情感传统或个人魅力，而是外在规范或利益交换，但可能逐渐发展出真情（“先婚后爱”）。

**例子：**《伪恋》中的一条乐与桐崎千棘，《总之就是非常可爱》中的由崎星空与司

```
这里必须严正吐槽一下《伪恋》
故事的开始是指两个黑帮家族缔结契约，这是典型的法理型
然后介绍小野寺是男主青梅竹马，到此为止，出现了一个法理型，一个传统型


而法理型和传统型是根本意义上矛盾的
法理型的逻辑：千棘和一条乐之所以扮演恋人，是一种功能性的关系，两者追求稳定，为了大局，感情必须让位于责任

传统型的逻辑：小野寺和一条的约定是基于童年羁绊的宿命论，这种关系是真实的、纯粹的

但是故事建立在为了和平必须演戏（法理），但是男主的核心驱动却是寻找真命天女（传统），如果男主重点在演习，那么就背叛了约定，如果重点在约定，就破坏了和平

说白了就是，你在高考，但你的注意力全在前面的黑丝小姐姐，思考着怎么才能得到小姐姐的微信


然后剧情最大的败笔出现了，作者登场了几个强魅力型角色，最具代表的就是万里花，她的登场直接夺走了所有观众的注意，至此，作者为了平衡，开始将所有角色转向魅力型（除了小野寺，她是本身数值太高）

千棘作为契约女友，却逐渐产生了魅力型的真爱。但剧情为了维持契约，强行让她因为傲娇而不敢承认，导致角色行为逻辑停滞不前

小野寺作为青梅竹马（约定之人），本该拥有最强的合法性。但作者为了平衡，不断削弱她的行动力（魅力型不足），让她变成了只会哭泣的背景板。这种青梅竹马的合法性因为缺乏魅力型的互动支撑，最终显得苍白无力


然后整个剧情最抽象的剧情出现，全员青梅竹马，至此，小野寺失去了合法性，因为就连她的钥匙都来自千棘的施舍，同时，这也让契约显得更加荒谬，既然大家都是一家人，为什么还要搞这种虚假的黑帮联姻
```

#### 留下来型权威
**核心逻辑：** 这个是极度内化的特殊法理型权威，逻辑来自于个人责任
**恋爱番表现：** 它的表现形式及其特殊，举个例子，比如说galgame（虽然galgame并不适用于这个框架），galgame的恋爱合法性是基于游戏本身在迁就你，你选A或者B，都是平等的，但是朋也的情况十分特殊，他完全可以离开小镇，渚也让他离开小镇，观众也能接受他离开小镇，但是最后，他还是选择了留下来

朋也并没有选择“最优解”，而是选择承担现实

## 总结
事实上，这个框架是可以用来分析绝大部分的恋爱合法性来源的，并且，这种框架还可以用于分析不同的国家之间的差异，举个例子

### 中国
#### 传统型
中国的传统型一般是与法理型相结合的产物，主要强调缘分天定，比如说《狐妖小红娘》

#### 魅力型
中国的魅力型主要集中于强调 强强联合 或者价值观共鸣，一般会和传统型结合，两者是知己好友

#### 法理型
中国的法理型主要集中于责任，大多是契约婚姻然后先婚后爱，比如说因为父母包办婚姻，两人被迫在一起，然后相处中产生情感


### 日本

#### 传统型
日本的传统型占比最大，主要强调从小到大的陪伴

#### 魅力型
这部分主要是追求瞬间共鸣以及角色的萌属性，比如说反差萌，还有一见钟情（我个人怀疑单纯是因为这样不需要讲述前因后果，只需要作为强设定推进就好了）

#### 法理型
日本的法理型大多是作为喜剧出现的，一般为传统型做铺垫（我个人猜测来自于日本集团主义的责任优先）


### 欧美

#### 传统型
欧美的传统型大多强调朋友到恋人，我个人不太了解详细的

#### 魅力型
这个部分是绝对核心，源于西方的个人主义和自由意志（这个我敢肯定），比如最经典的冰雪奇缘，怦然心动

# 冲突再生产
剧情靠什么不断制造张力，而不是告白即死

大体上可以区分为这五个类型

**博弈型：智斗、心理战、攻防转换**
**障碍型：身份、阶级、距离、疾病、外力阻隔**
**成长型：性格缺陷 → 改变 → 再确认关系**
**日常扰动型：重复但可变的生活事件**
**情绪余震型冲突：冲突不来自“现在”，而来自“已经发生的选择”，悔恨、内疚、否定不断再生产张力**

这里我就不详细展开来了，但我会用这个新框架分析原先的几部番存在的问题

#### 《辉夜大小姐》
恋爱关系的合法性来源是魅力型
两者的冲突关系则来自于博弈型

在冲突机制消失之后，作者并没有换一个新的冲突机制，仅仅是修改了合法性来源


#### 《高木》
恋爱关系的合法性来源是魅力型
两者的冲突关系来自于日常扰动型

从头到尾，作者都没有更换这两点
