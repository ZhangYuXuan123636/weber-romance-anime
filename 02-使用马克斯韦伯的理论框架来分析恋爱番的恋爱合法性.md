# 免责声明
这个框架问题很多很大，只是我随手写下来的，框架分为两部分，前半部分关系合法性来源只能被用于解释为什么两人会在一起（当恋爱作为问题时），后半部分则可以用于解释整部番剧（当恋爱作为问题时、当恋爱作为状态时）

尽管两个部分看起来完全不同，但是是高度相关的，在进行分析时可以拆分来看待，但是在作品内部，不能进行强行割裂


**框架的适用范围**
这个框架仅限于以恋爱为主线（主要冲突）或者有完整明确的恋爱故事的番剧

```
这里做一个特别声明
关于《败犬女主》，这部番从一开始就不是在竞争合法性，而是在讲一群人失去合法性之后的故事，她们不是输在最后，而是一开始就已经输了（当然了，如果哪天温水突然转性了，剧情开始转向恋爱，那另说）

关于《CLANNAD After Story》，也就是CLANNAD第二部中讲述的故事，在第一部中，朋也选择了渚，这个选择是必然性的，所以这部作品的本身讨论是恋爱作为状态的故事，本身是不适用这个框架的

但是我还是要特别强调一下，不单是这个框架的适用性的问题，也包含我个人对这部番的理解

我这个框架讨论的问题是恋爱是否成立、如何成立、能否持续作为叙事引擎
而《AS》讨论的是即便恋爱成立，人是否依然会被生活击垮

# 结论
这个框架实质上是在讨论两个问题
- 关系合法性的来源（为什么这两个人会在一起）
- 冲突再生产机制（一段恋爱关系如何被持续）

# 关系合法性来源
这原本是马克斯韦伯用于讲述政治权威合法性的，我对韦伯的框架做了泛化以及降维，将其用于分析动漫叙事结构和角色关系合法性，目的上存在差异政权追求稳定统治，故韦伯分类旨在解释统治的持续性。恋爱叙事追求戏剧张力，其“合法性”常作为冲突的起点或质疑的对象

这个分类本身是一种理想模型，在实践中一般是多种分类的混合展现

## 剧情内
剧情内的三种权威更多被用于逻辑上的合理，从剧情上，必须要一个合理的理由来将恋爱双方进行绑定，并且被剧情内的角色所接受

### 传统型权威
**核心逻辑：** 合法性源于“历来如此”的习惯与传统

**恋爱番表现：** 
青梅竹马从小一起长大，拥有共同的回忆和深厚的羁绊。这种恋爱的“合法性”不依赖于强烈激情，而是基于时间积淀的稳定感与熟悉感。


### 魅力型权威
**核心逻辑：** 合法性源于一方或双方超凡的个人魅力（如外貌、才华、性格等），产生非理性的情感

**恋爱番表现：**
角色在某个契机下相遇，因瞬间的心动或逐渐发现的独特魅力而坠入爱河。（多见于天降系）


### 法理型权威
**核心逻辑：** 合法性源于明确的规则、契约或理性计算，以及内在的承诺（后面会讲到中国法理型）

**恋爱番表现：** 
在恋爱番中，这种类型较少作为浪漫主线，但常以特殊形式出现，如政治联姻、假恋爱契约、相亲活动或恋爱游戏规则。关系的基础不是情感传统或个人魅力，而是外在规范或利益交换，但可能逐渐发展出真情（“先婚后爱”）





## 剧情外
剧情内部的三个分类与剧情外有些不同，剧情外的三种权威更多的是用于说服观众，让观众接受这两个人在一起是合理的，多用于党争

### 传统型权威
**核心逻辑：** 将过去产生的羁绊运用在现在

**恋爱番表现：** 
青梅竹马从小一起长大，拥有共同的回忆和深厚的羁绊。这种回忆相当于一种信息差，为什么传统型的角色会获胜呢？因为我了解你，我和你有过去的回忆，其他人没有




### 魅力型权威
**核心逻辑：** 合法性源于一方或双方超凡的个人魅力（如外貌、才华、性格等），产生强烈情感吸引。

**恋爱番表现：** 
我因为某个事物或者特质被你吸引，开始注意你，并且加深我们两人之间的情感，魅力型权威是最常见的，因为这种关系最容易发展故事，同时也最容易被观众所接受


#### 魅力再生产
在韦伯的魅力型权威中，魅力型权威必须不断通过“奇迹”或超凡表现来证明其合法性，否则会“例行化”为传统型或法理型

##### 可再生魅力
角色特质展现出的魅力是可再生的，比如说傲娇，面对不同的事件时，角色可以产生完全不同的傲娇反应

##### 不可再生
比如说告白、接吻，这些事件只有第一次或者在极少数事件的特殊场景下才能产生魅力（比如说，即便不是第一次接吻，死前最后接吻也是能产生魅力的）

```
我举两个反例，《辉夜》和《俄语艾莉》
为什么俄语艾莉会那么无聊，原因在于，两者的博弈是基于不变的规则，女主不知道男主会俄语，这种博弈会产生两个问题，其一：告白即死，因为这种信息差是唯一的叙事冲突，其二：不能告白，因为一旦告白，作者没有准备新的冲突，剧情就会卡死

《辉夜》的情况就好很多，同样是博弈，但是每次博弈都是基于完全不同的规则，但是目的是不变的，这就使得只要不表白，故事可以一直写下去
同时，这也是问题所在，前期的博弈之所以成立，是建立在两人完全对等的前提下（虽然设定上差距巨大，但是在学生会这个舞台上，两者只是学生，是完全对等的），当后面关于辉夜家族的故事展开，则完全变成了不对等的博弈，博弈的对手从白银御行本人，变成了四宫家的长老会。博弈的前提（双方自由意志的对决）已不存在
```




### 法理型权威
**核心逻辑：** 合法性源于明确的规则、契约或理性计算，以及内在的承诺（就比如后面会讲到的中国法理型）

**恋爱番表现：** 
在这个崇尚自由意志的时代，很少会将法理型权威作为增加恋爱合法性的实据，法理型大多发生在故事的开头或者结尾


#### 外化法理型权威
**核心逻辑：** 本质属于法理型权威的亚种，关系的推进源自于明确的规则，外部的契约
**恋爱番表现：** 
大多源自于某些小概率事件，导致两人产生了必然联系，比如说上下级、父母再婚

#### 内化法理型权威（存在主义法理型）
**核心逻辑：** 这个是极度内化的特殊法理型权威，逻辑来自于个人责任（我对个人责任的理解是，一种基于社会道德给自己施加的理性规则约束）
**恋爱番表现：** 
它的表现形式及其特殊，举个例子，比如说galgame（虽然galgame并不适用于这个框架），galgame的恋爱合法性是基于游戏本身在迁就你，你选A或者B，都是平等的，但是朋也的情况十分特殊，他完全可以离开小镇，渚也让他离开小镇，观众也能接受他离开小镇，但是最后，他还是选择了留下来

朋也并没有选择“最优解”，而是选择承担现实




```
这里必须严正吐槽一下《伪恋》
故事的开始是指两个黑帮家族缔结契约，这是典型的法理型
然后介绍小野寺是男主青梅竹马，到此为止，出现了一个法理型，一个传统型

故事建立在为了和平必须演戏（法理），但是男主的核心驱动却是寻找真命天女（传统），如果男主重点在演习，那么就背叛了约定，如果重点在约定，就破坏了和平

到此为止，故事没有问题，结果无非是在法理型和传统型二选一

然后剧情最大的败笔出现了，作者登场了几个强魅力型角色，最具代表的就是万里花，她的登场直接夺走了所有观众的注意，至此，作者为了平衡，开始将所有角色转向魅力型（除了小野寺，她是本身数值太高）

千棘作为契约女友，却逐渐产生了魅力型的真爱。但剧情为了维持契约，强行让她因为傲娇而不敢承认，导致角色行为逻辑停滞不前

小野寺作为青梅竹马（约定之人），本该拥有最强的合法性。但作者为了平衡，不断削弱她的行动力（魅力型不足），让她变成了只会哭泣的背景板。这种青梅竹马的合法性因为缺乏魅力型的互动支撑，最终显得苍白无力


然后整个剧情最抽象的剧情出现，全员青梅竹马，至此，小野寺失去了合法性，因为就连她的钥匙都来自千棘的施舍，同时，这也让契约显得更加荒谬，既然大家都是一家人，为什么还要搞这种虚假的黑帮联姻
```



```
辉夜大小姐是我不得不骂的一环，并且，我会将辉夜大小姐和高木同学做比较

辉夜大小姐在冰辉篇以前就是纯粹的魅力型，观众的重点就在于两个神进行博弈，双方都想让对方臣服于自己的魅力，白银的魅力来自于凡人之躯比肩神明，辉夜的魅力来自于强者的傲慢和掌控欲

韦伯指出，魅力型权威是不稳定的，它必须不断创造奇迹或胜利来维持。一旦魅力型领袖开始过“柴米油盐”的日子，其神性就会消失，这被称为“魅力的例行化”（这段的原意是用来讲述政权的，我借来用一下）

在告白成功之后，魅力型权威的根基——“博弈”彻底消失，对于一部建立在“傲娇博弈”基础上的番剧，这相当于自杀，作者失去了制造冲突的天然引擎

不过影响并没有显现出来，因为此时的剧情重点仍然在博弈上，包括后面的白银生病的那一段，此时的两者仍然是天才，即便有波动，也在观众的预期内

转折出现在毕业旅行那段，作者尝试跳出纯粹的魅力型叙事，原先的博弈变成了普通的离别焦虑，从这一段开始，后面的篇章持续走低

# 冲突再生产
剧情靠什么不断制造张力，而不是告白即死，并且这五种叙事引擎是可以混合使用的

大体上可以区分为这五个类型

**博弈型：智斗、心理战、攻防转换**
**障碍型：身份、阶级、距离、疾病、外力阻隔**
**成长型：性格缺陷 → 改变 → 再确认关系**
**日常扰动型：重复但可变的生活事件**
**情绪余震型冲突：冲突不来自“现在”，而来自“已经发生的选择”，悔恨、内疚、否定不断再生产张力**


# 附录
事实上，这种框架还可以用于分析不同的国家之间的差异，但是我没有数据做支撑，所以这里只是基于我的刻板印象做一种可能性的推测

### 中国
#### 传统型
中国的传统型一般是与法理型相结合的产物，主要强调缘分天定，比如说《狐妖小红娘》

#### 魅力型
中国的魅力型主要集中于强调 强强联合 或者价值观共鸣，一般会和传统型结合，两者是知己好友

#### 法理型
中国的法理型主要集中于责任，大多是契约婚姻然后先婚后爱，比如说因为父母包办婚姻，两人被迫在一起，然后相处中产生情感


### 日本

#### 传统型
日本的传统型占比最大，主要强调从小到大的陪伴

#### 魅力型
这部分主要是追求瞬间共鸣以及角色的萌属性，比如说反差萌，还有一见钟情（我个人怀疑单纯是因为这样不需要讲述前因后果，只需要作为强设定推进就好了）

#### 法理型
日本的法理型大多是作为喜剧出现的，一般为传统型做铺垫（我个人猜测来自于日本集团主义的责任优先）


### 欧美

#### 传统型
欧美的传统型大多强调朋友到恋人，我个人不太了解详细的

#### 魅力型
这个部分是绝对核心，源于西方的个人主义和自由意志（这个我敢肯定），比如最经典的冰雪奇缘

#### 法理型
我个人没看过这个类型的
